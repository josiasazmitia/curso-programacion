# Semana 05: JavaScript básico

## Objetivo

Aprender la lógica básica para manipular datos.

## Conceptos

- Variables
- Condicionales
- Funciones
- Arreglos y objetos

## Paso a paso

1. Copia el ejemplo en un archivo `app.js`.
2. Ejecuta el código en la consola del navegador.
3. Cambia los valores y observa resultados.
