# Semana 08: Portafolio final

## Objetivo

Empaquetar tu aprendizaje en un repositorio claro y compartible.

## Debe incluir

- Proyecto de biografía
- Proyecto estático tipo e-commerce
- Proyecto dinámico con API
- README claro con instrucciones

## Paso a paso

1. Organiza tus proyectos en carpetas.
2. Agrega capturas de pantalla.
3. Crea un README usando la plantilla.
4. Revisa ortografía y enlaces.
5. Publica en GitHub.
