# Semana 03: CSS para diseño limpio

## Objetivo

Aplicar estilos para mejorar claridad visual y orden.

## Conceptos

- Selectores
- Modelo de caja
- Flexbox
- Responsive básico

## Paso a paso

1. Crea un archivo `styles.css`.
2. Aplica estilos a tipografía y espaciado.
3. Usa Flexbox en una sección de tarjetas.
4. Haz commits por cada mejora visual.

## Mini reto

- Logra que las tarjetas pasen de 3 columnas a 1 en móvil.
