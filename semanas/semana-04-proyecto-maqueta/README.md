# Semana 04: Proyecto maqueta completa

## Objetivo

Construir un e-commerce estático completo, sin lógica de negocio todavía.

## Entregables mínimos

- Navbar
- Hero principal
- Lista o grid de productos
- Página de detalle
- Versión básica para móvil

## Paso a paso sugerido

1. Dibuja el wireframe en papel o en una app.
2. Crea la estructura HTML de la página principal.
3. Agrega estilos CSS.
4. Crea una segunda página de detalle de producto.
5. Conecta ambas páginas con enlaces.

## Mini reto

- Añade una sección de ofertas destacadas.
