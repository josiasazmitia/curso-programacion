# Semana 01: Fundamentos de Python

## Objetivo

Configurar un flujo básico en Python y entender variables, tipos de datos y operaciones simples.

## Conceptos

- Variables
- Tipos de datos (`int`, `float`, `str`, `bool`)
- Entrada y salida por consola
- Operadores básicos
- Conversión de tipos

## Cómo ejecutar

Desde la carpeta del repositorio, ejecuta:

```bash
python python/semana-01-fundamentos/ejercicios/01_variables.py
```

En Windows, también puede funcionar:

```bash
py python/semana-01-fundamentos/ejercicios/01_variables.py
```

## Paso a paso

1. Abre la carpeta `python/semana-01-fundamentos/ejercicios`.
2. Ejecuta cada archivo en orden.
3. Cambia valores para comprobar resultados.
4. Guarda tus cambios al finalizar cada ejercicio.
5. Haz commit y push al cerrar la sesión.

## Ejercicios

Crear un script que pida nombre, edad y ciudad, y luego imprima un resumen formateado con esos datos.

## Entregable

Al finalizar esta semana debes tener los 8 ejercicios ejecutables, con cambios personales en al menos 3 de ellos, y commits en GitHub.
