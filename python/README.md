# Ruta Python para principiantes

Esta ruta complementa el curso web con fundamentos de programación en Python.

Está pensada para avanzar paso a paso con lenguaje claro, útil para estudiantes y personas autodidactas en Guatemala y LATAM.

## Semanas

1. [Semana 01 - Fundamentos](semana-01-fundamentos/README.md)
2. [Semana 02 - Control de flujo](semana-02-control-flujo/README.md)
3. [Semana 03 - Funciones y colecciones](semana-03-funciones-y-colecciones/README.md)
4. [Semana 04 - POO básica](semana-04-poo-basica/README.md)
5. [Semana 05 - Archivos y JSON](semana-05-archivos-y-json/README.md)

## Recomendación de estudio

- Dedica entre 20 y 40 minutos al día a Python.
- Ejecuta al menos 2 ejercicios por sesión.
- Haz commits pequeños al terminar cada bloque.
