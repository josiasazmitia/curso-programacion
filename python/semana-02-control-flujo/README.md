# Semana 02: Control de flujo

## Objetivo

Aprender a tomar decisiones y repetir tareas usando condicionales y bucles.

## Conceptos

- `if`, `elif`, `else`
- Operadores lógicos (`and`, `or`, `not`)
- Bucle `while`
- Bucle `for`
- `break` y `continue`

## Cómo ejecutar

```bash
python python/semana-02-control-flujo/ejercicios/01_condicionales.py
```

## Paso a paso

1. Ejecuta los ejercicios en orden.
2. Cambia valores para probar casos diferentes.
3. Verifica qué condiciones se cumplen.
4. Ajusta mensajes y lógica.
5. Haz commit y push al terminar.

## Ejercicios

1. `01_condicionales.py`

## Mini reto

Crear un menú simple de consola con 3 opciones que se repita hasta que el usuario elija salir.

## Entregable

Al finalizar esta semana debes tener 8 ejercicios funcionando, al menos 1 mejora personal en un bucle o condición, y commits publicados.
