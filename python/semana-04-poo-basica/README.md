# Semana 04: POO básica

## Objetivo

Entender clases y objetos con ejemplos simples orientados a problemas reales.

## Conceptos

- Clase y objeto
- Atributos y métodos
- Encapsulación básica
- Composición simple entre clases

## Cómo ejecutar

```bash
python python/semana-04-poo-basica/ejercicios/01_clase_estudiante.py
```

## Paso a paso

1. Ejecuta los ejercicios 01 a 04 para entender clases base.
2. Revisa cómo se conectan `Product` y `Cart`.
3. Ejecuta ejercicios 05 a 08 para consolidar la lógica.
4. Modifica datos de prueba y vuelve a ejecutar.
5. Haz commit y push de cada bloque.

## Ejercicios

1. `01_clase_estudiante.py`
2. `02_estudiante_notas.py`
3. `03_clase_producto.py`
4. `04_clase_carrito.py`
5. `05_carrito_varios_productos.py`
6. `06_inventario_basico.py`
7. `07_mini_carrito_cli.py`
8. `08_mini_inventario_cli.py`

## Mini reto

Completar un mini proyecto CLI de carrito o inventario con menú simple y total final.

## Entregable

Al finalizar debes tener clases `Student`, `Product` y `Cart` funcionales, además de un mini CLI ejecutable commiteado y publicado.
