# Semana 05: Archivos y JSON

## Objetivo

Leer y escribir información en archivos para construir scripts útiles de consola.

## Conceptos

- Lectura de archivos `.txt`
- Escritura y agregado en archivos
- Formato JSON
- Carga y actualización de datos JSON

## Cómo ejecutar

```bash
python python/semana-05-archivos-y-json/ejercicios/01_leer_archivo_txt.py
```

## Paso a paso

1. Ejecuta ejercicios de texto primero (01 a 04).
2. Luego ejecuta JSON (05 a 07).
3. Cierra con el reto de registro de tareas.
4. Revisa archivos generados en la carpeta de ejercicios.
5. Haz commit y push de tu versión final.

## Ejercicios

1. `01_leer_archivo_txt.py`
2. `02_escribir_archivo_txt.py`
3. `03_append_archivo_txt.py`
4. `04_lectura_lineas.py`
5. `05_json_guardar.py`
6. `06_json_leer.py`
7. `07_json_actualizar.py`
8. `08_reto_registro_tareas.py`

## Mini reto

Crear un registro de tareas en JSON con opción de agregar y listar tareas desde consola.

## Entregable

Al finalizar debes tener scripts que creen y lean archivos `.txt` y `.json`, con resultados visibles y commits publicados.
