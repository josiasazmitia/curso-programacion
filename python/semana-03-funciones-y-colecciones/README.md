# Semana 03: Funciones y colecciones

## Objetivo

Organizar lógica en funciones y trabajar con listas, tuplas, sets y diccionarios.

## Conceptos

- Funciones
- Parámetros y retorno
- Alcance básico de variables
- Listas, tuplas, sets
- Diccionarios

## Cómo ejecutar

```bash
python python/semana-03-funciones-y-colecciones/ejercicios/01_funciones_basicas.py
```

## Paso a paso

1. Ejecuta cada ejercicio en orden.
2. Prueba valores distintos en funciones.
3. Agrega elementos en colecciones.
4. Revisa salida en consola.
5. Commit y push de avances.
