# Curso guiado: Programación web desde cero

Este repositorio es una ruta práctica para aprender programación web de forma progresiva. Está orientado a personas sin experiencia previa, con foco en construir proyectos reales semana a semana.

La idea principal es simple: aprender haciendo. Cada semana incluye una guía corta, un ejemplo mínimo y un mini reto para practicar. El curso no busca ser académico; busca que avances con confianza y resultados concretos.

Al finalizar, tendrás una base sólida en HTML, CSS y JavaScript, además de una rutina de trabajo con Git y GitHub para publicar tus avances.

## ¿Para quién es este curso?

- Personas que empiezan desde cero.
- Personas que ya intentaron aprender y quieren una ruta clara.
- Personas que prefieren práctica guiada en lugar de teoría extensa.

## Cómo usar este repositorio

1. Empieza en [Semana 01](semanas/semana-01-setup/README.md).
2. Lee el README de la semana completa.
3. Ejecuta o replica el ejemplo indicado.
4. Completa el mini reto.
5. Haz commits pequeños y sube tus cambios a GitHub.

Flujo recomendado cada sesión:

```bash
git status
git add .
git commit -m "docs: semana 01 guía"
git push
```

Ejemplos de mensajes de commit:

- `docs: semana 01 guía`
- `feat: agregar biografía HTML`
- `fix: corregir enlace interno`

## Ruta de aprendizaje

### Mes 1: Web estática

- Semana 01: Setup, Git y primera web.
- Semana 02: Estructura HTML semántica.
- Semana 03: Estilos con CSS y responsive básico.
- Semana 04: Maqueta completa tipo e-commerce.

### Mes 2: Web dinámica

- Semana 05: Lógica base con JavaScript.
- Semana 06: Interacción con DOM y eventos.
- Semana 07: Consumo de API con `fetch`.
- Semana 08: Portafolio y presentación final en GitHub.
